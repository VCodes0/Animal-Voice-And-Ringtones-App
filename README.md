<h1>🐾 Animal Voice & Ringtones App</h1>

<p>
  A Flutter-based mobile app that lets users browse, play, and set ringtones of animal sounds.
  Built for smooth performance and modern UI using <code>GetX</code>, <code>Provider</code>, and <code>just_audio</code>.
</p>

<hr>

<h2>📱 Features</h2>
<ul>
  <li>🎵 Play high-quality animal sounds</li>
  <li>🔊 Set any sound as a ringtone (platform-dependent)</li>
  <li>🖼️ Load and cache animal images using <code>cached_network_image</code></li>
  <li>⚙️ Lightweight and reactive with <code>GetX</code> and <code>Provider</code></li>
  <li>🌐 Works offline after initial asset load</li>
</ul>

<h2>🛠️ Built With</h2>
<ul>
  <li><strong>Flutter</strong> (Dart)</li>
  <li><a href="https://pub.dev/packages/get">getx</a> – State management & navigation</li>
  <li><a href="https://pub.dev/packages/provider">provider</a> – Dependency injection</li>
  <li><a href="https://pub.dev/packages/just_audio">just_audio</a> – Audio playback</li>
  <li><a href="https://pub.dev/packages/cached_network_image">cached_network_image</a> – Image caching</li>
</ul>

<h2>🚀 Getting Started</h2>

<h4>Prerequisites</h4>
<ul>
  <li>Flutter SDK (3.0 or higher)</li>
  <li>Android Studio or VS Code</li>
  <li>Physical or virtual Android device</li>
</ul>

<h4>Installation</h4>
<pre><code>git clone https://github.com/VCodes0/Animal-Voice-And-Ringtones-App.git
cd Animal-Voice-And-Ringtones-App
flutter pub get
flutter run</code></pre>
